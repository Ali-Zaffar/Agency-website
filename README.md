# Modern Agency Website
## 📖 Project Overview

This project is a modern agency website converted from a Figma design into a fully responsive WordPress website using the Kadence Theme and Gutenberg Editor.
The website focuses on clean design, fast performance, SEO optimization, and an excellent user experience.
## 🚀 My Role

- Converted Figma design to WordPress
- Built pages using Gutenberg Blocks
- Customized Kadence Theme
- Responsive Design
- SEO Optimization
- Performance Optimization
